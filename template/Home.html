Introduction

The Project “Pet Connect”, was developed using the technology Python Language. The aim of this is to outline the design and development of a Pet Connect Portal. The main objective of this project is to provide multiple services for the users from purchasing a pet to maintaining a pet. It provides the information and resources on pet care, tips and advices for pet owners. This acts as a shopping website where pets can be sold. The features of this are Training, Handling and Helping the pets, these helps the pet owners. This can be used by the pet owners to find accurate information on pet’s nutrition and health. This mainly focus on dog, cat and several pet categories like rabbit, pigeon etc. There exist five modules, they are Customer, Admin, Handler, Helper and Trainer. The main objective of the project is to provide multiple services in one platform which makes easy to the User. All the required things can be provided to the user at one place, through this the user can easily access their requirements. This Project is making it easy to access for the Users.

Existing System

Generally, We Research before buying a pet and make sure that pet is the right choice of pet for us. We gather information from google, friends, relatives and etc. Not only for purchasing a pet, for its maintenance also we take suggestions from different resources. We usually visit pet shops for purchasing pets. They provide services like grooming, training and maintenance etc. For pets Health care we visit veterinary hospital and consult doctor. For each and every pet’s need we should move to different places. For handling and helping pets we need to search more.

Disadvantages

• This is huge process, consumes more time.
• We have to visit multiple places for pet needs.
• Because of multiple suggestions we may not sure about the decisions.
• Lack of Information.

Proposed System

Pet Connect is a portal where we can have multiple services at one platform, from selling to training the pets including helping and handling the pets. This portal acts as a shopping website. In this portal we can buy the pets from the list provided. We can know clear information about pets like their breed, characteristics, cost and etc. To train the pets, this portal provides the list of trainers and their performance ratings based on that we can contact them. Through this we can contact with helpers and handlers for pets. We can get detailed information regarding pets in this portal.

Advantages

• It is a single platform where we get all the related information of pets.
• It provides detailed information about the service.
• This portal helps to finish the process easily.
• Client can raise the ticket if he has any issue with the service.
